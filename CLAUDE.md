# JustGoodCampers

## Project Overview
Family-owned camper rental business based in New Zealand. The site lets international travellers browse, learn about, and book one of two camper types for their NZ road trip.

## Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **CMS:** Sanity (planned)
- **Database/Auth:** Supabase (planned)

## Camper Types
- **2-Berth Compact** — for couples or solo travellers (`/campers/2-berth`)
- **4-Berth Family** — for families or groups (`/campers/4-berth`)

## Pages
| Route | Description |
|-------|-------------|
| `/` | Homepage |
| `/campers` | Camper overview |
| `/campers/2-berth` | 2-Berth Compact detail |
| `/campers/4-berth` | 4-Berth Family detail |
| `/whats-included` | What's included with a rental |
| `/faq` | Frequently asked questions |
| `/blog` | Blog overview |
| `/blog/[slug]` | Blog post detail |
| `/contact` | Contact page |
| `/book` | Booking flow |

## Brand & Audience
- **Target audience:** International travellers visiting New Zealand
- **Brand values:** Simple, trustworthy, personal, no-nonsense
- **Tone:** Friendly and direct — no jargon, no fluff
- **Language:** English (primary), NZ market
